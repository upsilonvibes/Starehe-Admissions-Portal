# Starehe-Apply-System 🇰🇪

A modern, React-based enrollment portal designed to digitize the application process for **Starehe Boys' Centre (Yellow Form)** and **Starehe Girls' Centre (Blue Form)**. This system streamlines student data collection, pathway ranking, and school selection for Grade 10 transitions and transfers.

## 🚀 Features

*   **Dynamic Portal Selection**: Tailored UI for both SBC (Yellow) and SGC (Blue) institutions.
*   **3-Choice Priority System**: Allows students to rank their preferred Pathways (STEM, Social Sciences, Arts) and specific Tracks in order of priority.
*   **Intelligent Validation**: 
    *   Automatic resetting of tracks when a pathway is changed.
    *   Conditional logic for Transfer Students (Current Grade & Reason for Transfer).
*   **Vertical UX**: Optimized form layout for readability and professional data entry.
*   **Backend Connectivity**: Integrated status check to verify Node.js server availability.

## 🛠️ Tech Stack

*   **Frontend**: React.js (Vite)
*   **State Management**: React Hooks (`useState`, `useEffect`)
*   **Styling**: Pure CSS3 (Flexbox architecture)
*   **Backend**: Node.js (planned integration for data persistence)

## 📁 Project Structure

- `/src/App.jsx` - Core application logic and form state.
- `/src/App.css` - Custom vertical grid system and portal branding.
- `/public/images_starehe/` - Contains institutional branding and reference logos.

## 🏁 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/upsilonvibes/Starehe-Apply-System.git](https://github.com/upsilonvibes/Starehe-Apply-System.git)
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
